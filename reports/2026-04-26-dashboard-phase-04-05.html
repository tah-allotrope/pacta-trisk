03 / Math / Algorithm Used

Math / Algorithm Used

No new TRISK model math was introduced. The dashboard continues to treat the frozen snapshot as the source of truth and only adds presentation logic.

The TRISK page derives:

  • NPV change percent from the stored npv_change column for chart readability.
  • PD basis-point deltas by multiplying the stored decimal pd_change by 10,000.
  • Sensitivity filtering by subsetting the precomputed sensitivity_results.csv grid on one parameter and one value at a time; no recomputation occurs.

Limitations / Second-Best Alternative
