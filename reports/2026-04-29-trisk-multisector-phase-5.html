06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • automotive remains intentionally deferred, so the multisector story is complete only for power, cement, and steel.
  • cement and steel still present sector-level SDA context rather than borrower-specific SDA alignment, which limits how strongly the dashboard can claim borrower-level combined alignment-and-stress evidence.
  • The Streamlit smoke check was manual; it confirms the visible workflow, but it is not yet a fully scripted browser regression.
  • Second-best alternative: leave the rerun knowledge distributed across phase notes only and rely on operator memory. This was rejected because the multisector flow now spans five generation/publish commands plus two verification steps, and undocumented sequencing would make future refreshes fragile.
