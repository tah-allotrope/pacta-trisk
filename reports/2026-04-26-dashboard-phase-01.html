03 / Math / Algorithm Used

Math / Algorithm Used

Hosting Decision Scoring: Each platform was scored on 8 criteria (card required, sleep behavior, RAM/CPU, file-size limit, custom domain, CSV/PNG hosting, secrets support, build effort). Streamlit Community Cloud won on the highest composite score for the read-only demo use case — no database, no Docker, no card required.

Data Contract: File selection followed a simple inclusion rule: every structured output from synthesis_output/ named with alignment, SDA, trajectory, or sensitivity semantics is included. Raw pipeline intermediates (01_vn_matched_raw.csv, 04_vn_ms_company.csv — note this IS included) are included where they add decision-useful detail. Temp files and debug outputs are excluded.

06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • Static snapshot: dashboard/data/ is a point-in-time copy. Future pipeline re-runs require scripts/refresh_dashboard_data.R to be executed manually. No automatic sync.
  • HTML reports not rendered inline yet: Reports are copied to dashboard/data/reports/ but the Streamlit app (PHASE-02+) is needed to serve them. PHASE-01 is purely the data layer.
  • Second-best alternative: Hugging Face Spaces (Streamlit SDK). Nearly identical developer experience, same codebase, also free / no card. Kept as documented fallback.
